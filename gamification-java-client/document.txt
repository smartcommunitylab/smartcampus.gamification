## Getting Started

### Basic Auth
In order to add or modify API, it is required to edit the api-docs-basic.json file on project root before generating the sdk client stub running the following command from project root(gamification-java-client). 

```shell
java -jar lib/swagger-codegen-cli.jar generate -DhideGenerationTimestamp=true -i api-docs-basic.json -l java --api-package it.smartcommunitylab.basic.api --artifact-id gamification-java-client --model-package it.smartcommunitylab.model --invoker-package it.smartcommunitylab.basic 
```
